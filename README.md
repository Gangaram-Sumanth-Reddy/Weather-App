# Global Weather App

A modern and lightweight **Weather Application** that allows users to check real-time weather conditions for any city worldwide.
The application fetches live weather data using a Weather API and presents it through a clean and responsive user interface.

🔗 **Live Demo**
https://weather-app-theta-one-99.vercel.app/

---

## Overview

The Global Weather App provides quick access to essential weather information including temperature, humidity, wind speed, and weather conditions. The interface is designed to be minimal, responsive, and easy to use across all devices.

---

## Key Features

• Real-time weather information for any global location
• Displays temperature, humidity, wind speed, and weather conditions
• Responsive design compatible with desktop, tablet, and mobile devices
• Light mode and dark mode toggle for improved user experience
• Clean and simple user interface for quick weather searches

## Getting Started

### 1. Clone the Repository

```
git clone https://github.com/Gangaram-Sumanth-Reddy/Weather-app.git
```

### 2. Navigate to the Project Directory

```
cd Weather-app
```

### 3. Run the Application

Open the `index.html` file in your browser.

---

## Usage

1. Enter the name of a city in the search field
2. Click the **Get Weather** button
3. The application will display the current weather details for that location
